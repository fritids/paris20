Conditions d'utilisation

Données personnelles

Cette application vous permet de signaler des problèmes dans le XXème arrondissement. Les données collectées à cette occasion sont :

  • Le type de problème
  • Votre description de ce problème

Vos numéro de téléphone, nom, adresse IP ne sont pas collectés.

Modération

Les modérateurs de cette application se réservent le droit de modérer les descriptions de problème. Les causes d'une telle modération sont par exemple des messages diffamatoires ou incitant à la haine.

Toute communication relevant d'une forme de délation sera aussi modérée.
